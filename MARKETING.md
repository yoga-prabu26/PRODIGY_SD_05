## 2. Project Description (for GitHub README header / portfolio)

WebScrapeX Pro is a full-featured desktop data intelligence platform that
transforms raw website product listings into structured, analyzable data.
Built with an MVC-inspired architecture, it combines a multi-threaded scraping
engine, a SQLite-backed persistence layer, a Matplotlib analytics dashboard,
and a polished CustomTkinter interface with dark/light theming — delivering an
experience comparable to commercial data extraction SaaS tools.

---

## 3. Key Learnings

- Designing a modular, OOP-based Python application following an MVC-style
  separation of concerns (Model: scraper/database/analytics/exporter, View/Controller: UI)
- Implementing multi-threaded background tasks in a Tkinter-based GUI to keep
  the interface responsive during long-running operations
- Building a robust web scraping pipeline with URL validation, retry logic,
  and graceful error handling using `requests` and `BeautifulSoup4`
- Designing a relational SQLite schema to persist scraping sessions and
  product records with foreign-key relationships
- Generating dynamic, theme-aware data visualizations with Matplotlib embedded
  directly inside a CustomTkinter window
- Implementing data export pipelines to CSV, Excel (with auto-formatting via
  OpenPyXL), and JSON using Pandas
- Applying professional UI/UX principles: sidebar navigation, KPI cards,
  sortable/filterable/paginated tables, and consistent dark/light theming
